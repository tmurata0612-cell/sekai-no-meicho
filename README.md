# 世界の名著

著者本人がゲストに来る対談ラジオで、世界の名著の思考フレームを「読んだことがある」ではなく **自分の判断や会話で使える状態** にするための PWA。

MC シオリと懐疑役アラタが、各巻の著者本人を招いて対話する 10〜15分のラジオ番組を、事前生成の音声付きで配信する。1冊=1シリーズ、各話に台本・まとめノート・タップ式クイズを備える。

初回リリースはプロトタイプ5冊22話:

|---|---|---|
| 君主論 | ニッコロ・マキャベリ | 6 |
| 孫子 | 孫武 | 4 |
| 生物から見た世界 | ヤーコプ・フォン・ユクスキュル | 3 |
| イワン・イリイチの死 | レフ・トルストイ | 4 |

## 使い方（ローカル）

音声(MP3)は容量の都合でリポジトリには入れず、GitHub Releases（タグ `audio-v1`）に置いている。

- **公開版**: GitHub Pages でそのまま動作（`config.js` の `AUDIO.mode = "release"`）。音声は Releases の絶対URLから再生。
- **ローカル検証**: `config.js` の `AUDIO.mode` を `"local"` にし、`世界の名著/`（このリポジトリの1つ上の作業ツリー）をルートに `python -m http.server 8777` → `http://localhost:8777/app/`。音声は `../audio_out/` から読む。

## 素材のクレジットとライセンス

- **音声合成**: [VOICEVOX](https://voicevox.hiroshiba.jp/)。利用規約に基づき、使用話者のクレジットをアプリ設定画面に表示している（九州そら／白上虎太郎／玄野武宏／青山龍星／麒ヶ島宗麟／雀松朱司）。
- **著者肖像**: すべて Wikimedia Commons のパブリックドメイン素材を1点ずつライセンス確認のうえ採用（トリミング・リサイズのみ）。詳細は [`assets/portraits/CREDITS.md`](assets/portraits/CREDITS.md)。同時代肖像が無い／PD性を確定できない著者（孫武・ユクスキュル）と架空のMC・懐疑役は自作SVG。
- **本文・番組訳**: 原典はパブリックドメインのものを用い、番組独自に翻訳・構成したオリジナル。既存の現代語訳・翻訳（保護期間中）は文言を参照・借用していない。

肖像の追加・差し替え時は、必ず各画像のライセンスをファイルページで1点ずつ確認し、CREDITS とアプリ設定画面のクレジットを更新すること。
