### 調査で判明した落とし穴（次に肖像を足す人へ）

1. **Commons API / 概要欄の `LicenseShortName` を信用しない。必ず生wikitext（`?action=raw` か `Special:Export`）を読む。** 集約されたライセンス欄は**作品の層だけ**を報告し、**撮影者の権利を隠す**ことがある。実例＝`File:Plato Ny Carlsberg Glyptotek IN2553.jpg` は API 上 `"Public domain"` と出るが、それは彫像（`PD-old-100-1923`）の話で、**写真は `{{self|Cc-by-4.0}}`**。これを見落とすとCC-BYの写真をPDとして公開してしまう。
2. **「彫像の写真」と「絵画の複製」は別物。** `PD-Art` は**2次元作品の忠実な複製にしか適用できない**。3次元の胸像を撮った写真には撮影者の著作権が生じうるので、`PD-Art`/`PD-old-100` **だけ**を根拠にした胸像写真は採用しない（タグの適用自体が誤り）。採れるのは `{{PD-self}}` 等、**撮影者本人の明示的なPD献呈**がある場合だけ（＝`16-thucydides` はこれで通した）。プラトンの胸像は8点すべてCC-BY/CC-BY-SAで全部却下し、代わりに**ラファエロ「アテナイの学堂」＝2次元のフレスコ**（`|medium={{fresco}}`）を使うことで問題ごと回避した。
3. **無名の写真は「撮影年」ではPDにならない。** 起算点は**最初の公表年**（EU指令2006/116 Art.1(3)・日本の著作権法52条）。初公表年が確定できない写真は、どれだけ古く見えても採用しない（＝カフカ・デュルケーム・ポアンカレはこれで落ちた）。**被写体の没年は根拠にならない。**
